# ⚡ P2 - Pokémon (React Web)

Projeto final da disciplina de React da Residência TIC/Software do Serratec.

Aplicação web desenvolvida em React com integração à API de Pokémon, permitindo consultar e visualizar informações dos Pokémons de forma dinâmica e interativa.

- React

- Axios
- CSS
- Pokémon API

## ✨ Funcionalidades

- Listagem de Pokémons
- Visualização de detalhes
- Navegação entre páginas
- Consumo de API externa
- Componentização com React
- Gerenciamento de estados
- Login e cadastro do usuário
- Cadastrar novo Pokémon
- Pokedéx que armazena pokémons capturados

## 💻 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/albino57/P2-Pokemon.git
````

Instale as dependências:
```bash
npm install
npm install @google/model-viewer
````

Execute o projeto:

````
npm run dev
````

O projeto será iniciado no navegador.
